# 压缩包密码破解
This repository provides  codes to crack archive with password
## 主要功能
通过暴力枚举的方式，破解压缩包密码，后期将搭建md5-password共享平台。
## 版本迭代
### 0.1.0.0
支持内置字典以及外部字典破解</br>
支持将内置字典导出至外部文件</br>
通过多进程读写文件方式提速</br>
简单的UI窗口</br>
